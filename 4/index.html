Locality-preserving essence vector (LPEV) model

    Given training pairs from neighboring-paragraph set $\mathbb{D}^+$ and faraway-paragraph set $\mathbb{D}^-$, and vocabulary set $\mathbb{V}$,

  • $P_{D_t}$ ($|\mathbb{V}|$-dim): frequency count of a word/ term in $D_t$ (normalized to unit-sum)
  • $P_{BG}$ ($|\mathbb{V}|$-dim): frequency count of a word/ term in $\mathbb{D}$ (normalized to unit-sum)

Component: fully connected multilayer NN

  • Paragraph encoder: \[ f(P_{D_t}) = v_{D_t} \]
  • Background encoder: \[ g(P_{BG}) = v_{BG} \]
  • Decoder: \[ h(\alpha_{D_t}*v_{D_t}+(1-\alpha_{D_t})*v_{BG}) = P_{D_t}' \]

    Monte Carlo Tree Search

    A simulation-based search algorithm that converges to a solution to the planning problem

    MCTSnet

    Initialize simulation time $t = 0$ and current node $s_0 = s_A$

    Repeat the following steps for M times

    1. Forward simulation from root state
    2. Sample action based on simulation policy, compute reward and next state, increment $t$
    3. Evaluation final leaf node using the embedding network
    4. Back-up using the back-up network

    Readout network outputs a action from the root embedding

    Network Architecture

    • Redidual block: 2 64 channel 3x3 conv
    • Embedding network: 64 channel 3x3 conv --> 3 residual blocks --> 32 channel 1x1 conv --> linear layer (128 dim)
    • Simulation network: 32 channel 3x3 conv --> 2 residual blocks --> 32 channel 1x1 conv + MLP with single hidden layer (128 dim)
    • Readout network: MLP with single hidden layer (128 dim)
